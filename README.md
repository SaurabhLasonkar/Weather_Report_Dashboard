# Weather_Report_Dashboard
# 🌦️ Weather Dashboard (7-Day Analysis)

## 📌 Project Overview

This project is a **Weather Dashboard** that provides insights and analysis of the **past 7 days weather data**. The data is fetched using a **Weather API**, processed, and visualized to help users understand trends like temperature, humidity, and weather conditions.

---

## 🚀 Features

* 📊 7-day historical weather analysis
* 🌡️ Temperature trends visualization
* 💧 Humidity and weather condition tracking
* 📍 Location-based weather insights
* 📈 Interactive dashboard (Power BI / Python / etc.)
* 🔄 Real-time data fetching using Weather API

---
